Bulk sending

To send a batch of messages at once:

  1. Go to the template page and create a new template (or choose an existing one).
  2. Select Send.
  3. If you’re sending emails, select Upload a list of email addresses. If you’re sending text messages, select Upload a list of phone numbers.
  4. You will then be propted to upload CSV a list of email addresss or phone numbers, as well as the details of any placeholders your templates uses (there is an example CSV you can download).
